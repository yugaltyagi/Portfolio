# Yogeshwar Tyagi — Portfolio

Modern portfolio website showcasing education, experience, skills, and GitHub projects.

## Live site

**https://yugaltyagi.github.io/Portfolio/**

## Local preview

Open `index.html` in a browser, or run a simple server:

```bash
npx serve .
```

## GitHub Pages setup

1. Push this repo to `github.com/yugaltyagi/Portfolio`
2. Go to **Settings → Pages**
3. Under **Build and deployment**, set **Source** to **GitHub Actions**
4. Push to `main` — the workflow in `.github/workflows/deploy.yml` deploys automatically
